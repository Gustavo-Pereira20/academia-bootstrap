# SharkFit 🦈💪

Landing page moderna e responsiva para a academia **SharkFit**, desenvolvida com foco em apresentação institucional, planos da academia e contato com clientes.

---

# 📋 Funcionalidades

* ✅ Navbar responsiva com dropdown
* ✅ Hero section (seção de destaque)
* ✅ Cards de apresentação dos planos
* ✅ Galeria de imagens da academia
* ✅ Formulário de contato
* ✅ Rodapé
* ✅ Alertas personalizados com SweetAlert2
* ✅ Layout totalmente responsivo

---

# 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap
* SweetAlert2

# 🖥️ Seções da Página

## 🔹 Navbar Responsiva

Menu de navegação adaptável para dispositivos móveis, contendo dropdown para melhor experiência do usuário.

## 🔹 Hero Section

Seção principal com destaque visual da academia, incentivando novos alunos a conhecerem a SharkFit.

## 🔹 Planos da Academia

Cards modernos apresentando os planos disponíveis e seus benefícios.

## 🔹 Galeria de Imagens

Exibição de fotos da academia, equipamentos e ambiente.

## 🔹 Formulário de Contato

Área destinada ao envio de mensagens e solicitações de contato.

## 🔹 Rodapé

Informações complementares e direitos autorais.

## 🔹 SweetAlert2

Alertas personalizados para melhorar a interação do usuário.
